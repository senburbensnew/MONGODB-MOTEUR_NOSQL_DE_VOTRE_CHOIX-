### Pour les sources java nous avons cree un projet MAVEN et utilise IntelliJ comme IDE
### Pour utiliser il faut telecharger IntelliJ et ouvrir le projet maven "cassandra_java" a partir de celui-ci
### A l'ouverture du projet l'IDE devrait telecharger les dependances et les intaller automatiquement a partir du fichier POM.xml\
### Si tel n'est pas le cas vous pouvez faire ces manipulations manuellement
